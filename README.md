#MAPA DE TEMPERATURAS EM PYTHON

##OBJETIVO

O projeto tem como objetivo ler e analisar temperaturas fornecidas pelo usuário em diferentes horários do dias, durante os cinco dias úteis da semana. O sistema fornece relatórios sobre os dados fornecidos.

##COMO USAR?

O sistema funciona no terminal de qualquer compilador de python, é necessário ter python instalado no dispositivo
